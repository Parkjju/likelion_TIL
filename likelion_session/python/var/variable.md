# 변수와 상수

-   데이터
    -   변할 여지가 有 -> 변수
    -   변할 여지가 無 -> 상수

# 입력과 출력

-   출력 -> 출력을 명령하는 함수가 있을 때 터미널 상에 출력
-   문자열 출력시 큰따옴표 필요

```python
print(1+2)
```

-   변수의 이름 지정, 대입연산자를 통해 값을 대입

```python
number = 5

print(number)
```

    -   숫자를 입력하고싶으면 변환해주기
    -   `a = int(input("숫자를 입력해요: "))`
