# Bootstrap

### 부트스트랩이란?

-   CSS/JavaScript기반 웹 프레임워크

    -   **웹 프레임워크** = '웹을 만드는 재료들의 모음'

-   반응형 웹 지원
-   브라우저 호환성

### 이용하기

-   부트스트랩 CDN - CDN링크를 적용할 html head태그에 복사
-   JQuery 코드를 가져와야함 (3.x 버전, umcompressed)
-   [jquery 링크](http://code.jquery.com/)

-   콘테이너 - 브라우저 내에 여백 디자인

```html
<div class="container"></div>
```
