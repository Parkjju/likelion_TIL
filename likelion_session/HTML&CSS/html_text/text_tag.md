# 텍스트와 관련된 태그

-   제목 태그 - 중요도에 따라 h1~h6

    -   body태그 안에서 태그로 감싸지 않은 문서 -> 기본적으로 태그로 싸인 것처럼 작동

-   본문 태그 - paragraphs

    -   줄바꿈 적용 - br태그 이용
    -   종료 태그가 없는 br -> 빈 요소
    -   pre태그 - 적은 내용대로 표시 (줄바꿈도 해줌)

-   글자와 관련된 태그
    -   strong (볼드체)
    -   em태그 (이탤릭체)
    -   sub태그 (아래에 기입) - 로그
    -   sup태그 (위에 기입) - 지수
    -   ins태그 - 밑줄
    -   del태그 - 취소선
